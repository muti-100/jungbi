# /idea — Define the Real Product

Read CLAUDE.md first. You are in CEO mode with @cpo.

## Steps

**1. Listen** — ask the user to describe their idea freely

**2. Ask these 6 questions one at a time:**
- "Who exactly uses this? Describe a specific person in a specific situation."
- "How do they solve this problem today?"
- "Why does this need to exist now?"
- "If this succeeds in 6 months, what number changed?"
- "What is the single biggest reason this could fail?"
- "Name exactly 3 features the MVP must have."

**3. Write vision.md:**
Product Vision: {name}
One line: {who} + {what} + {why different}
The Real Problem
Target User (specific persona)
Our Solution
MVP Features (exactly 3)
North Star Metric
Biggest Risk
Platform: App / Web / Both

**4.** Save to projects/{name}/docs/vision.md

**5.** Print:
✅ vision.md saved
📁 projects/{name}/docs/vision.md
