# Product Vision: 정비나라 (jungbi)

**One line:** 도시정비사업 관계자가 재개발/재건축/소규모주택정비/모아주택 절차를 법령 기반으로 한눈에 관리하고, 법 개정 시 자동 갱신되는 SaaS 플랫폼

## The Real Problem

정비사업은 **절차법** — 법에 정해진 절차만 따라가면 된다. 그런데:
- 관련 법령이 3개 이상(주거환경정비법, 빈집및소규모주택정비에관한특례법 등)에 흩어져 있음
- 시도별 조례가 다 다르고, 수시로 개정됨
- 국가법령정보센터에서 직접 검색해야 하는데, 내 상황에 맞는 법령을 직관적으로 찾기 어려움
- 법무법인 의뢰 비용이 높고, 유사 사례를 찾기도 힘듦

**핵심 인사이트:** 절차법이므로 절차 자동화가 가능하다 → 정비사업전문관리업 자체를 자동화할 수 있다.

## Target Users

도시정비사업 관련 전체 이해관계자:
- **정비사업전문관리업체** — 여러 조합의 사업 절차를 동시에 관리
- **정비사업조합 집행부** — 우리 조합의 현재 단계와 다음 할 일을 파악
- **정비사업 컨설팅** — 고객에게 정확한 절차/법령 근거 제공
- **도시정비 시공사** — 입찰/시공 단계별 법적 요건 확인
- **시도 공무원** — 인허가 심사 시 법령/조례 교차 확인
- **도시정비 변호사** — 소송/자문 시 법령 근거 빠른 조회

**대표 페르소나: 박 과장 (38세, 정비사업전문관리업체, 서울)**
- 동시에 5개 조합의 재개발/재건축 사업 관리 중
- 매주 법령 개정 여부를 국가법령정보센터에서 수동 확인
- 시도별 조례 차이를 엑셀로 정리하고 있으나 갱신이 밀림
- "법 바뀐 거 모르고 절차 진행했다가 사업 지연되면 큰일"

## Our Solution

### 전체 기능 (Full Scope)

1. **법령/조례 통합 정리 뷰** — 주거환경정비법, 빈집및소규모주택정비에관한특례법을 사업 유형별(재개발/재건축/소규모/모아주택)로 구조화하여 표시
2. **시도별 조례 비교** — 동일 절차에 대한 각 시도 조례 차이를 테이블로 비교
3. **절차 플로우차트** — 사업 유형별 전체 절차를 시각적 플로우차트로 한눈에 표시
4. **법령 자동 갱신/알림** — 국가법령정보센터 크롤링으로 법 개정 감지 시 자동 업데이트 + 알림
5. **조합별 절차 진행 관리 (SaaS)** — 조합마다 현재 어느 단계인지 추적, 다음 단계 안내
6. **일정 관리** — 절차 단계별 법정 기한/예상 기한 관리, 캘린더 뷰
7. **유사 사례 검색** — 법령/판례/행정해석 기반 유사 사례 검색

## Revenue Model

**조합별 SaaS 구독** — 월 정기 구독으로 수익화
- 조합이 본인들에게 맞는 절차 관리 및 일정 관리 가능

## North Star Metric

**유료 구독 조합 수 200개** (6개월 후 목표)

## Biggest Risk

- **법령 데이터의 정확한 갱신이 안 되면 신뢰를 잃음**
- **개별 조합 데이터 제공이 부정확하면 이탈**
- 완화 전략: 국가법령정보센터 API/크롤링 자동화 + 법무 검수 파이프라인

## Platform

**Web** (반응형) — Next.js 풀스택

## Tech Stack

- Frontend: Next.js 14 (App Router) + TypeScript + Tailwind CSS
- 시각화: React Flow (절차 플로우차트), FullCalendar (일정)
- Backend: Next.js API Routes + Supabase (DB + Auth + Realtime)
- 법령 크롤링: Python worker (국가법령정보센터 Open API)
- 검색: Supabase Full-Text Search / pgvector (유사 사례)
- 배포: Vercel + Supabase Cloud
