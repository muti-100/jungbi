# Product Vision: Waruru

## One Line
오프라인 만남이 어색한 사람들을 위한 실시간 GPS 매칭 플랫폼 — 앱이 장소까지 잡아주고, 만남 후 기록까지 남긴다.

---

## The Real Problem
초연결 사회에서 역설적으로 오프라인 만남의 비중은 줄었다.
사람들은 만나고 싶지만 — 빌미가 없고, 시간이 없고, 새로운 만남에 대한 심리적 허들이 높다.
기존 서비스(틴더, 당근 모임, STST, Timeleft)는 "매칭"은 해주지만 "실제로 만나게" 하는 마찰을 제거하지 못했다.

---

## Target User (Persona)
**"바쁜 도시 직장인, 28세 김지수"**
- 퇴근 후 강남 근처에 있지만 혼자
- 틴더는 질렸고, 번개모임은 어색하고, 클럽은 싫다
- 취향이나 공통 관심사가 딱히 없어도 그냥 사람이 만나고 싶다
- 새로운 만남에 대한 심리적 부담 때문에 결국 집에 간다

**공통 심리:**
- 만날 시간은 없지만 만나고 싶다
- 만날 빌미가 필요하다
- 새로운 오프라인 놀거리가 필요하다
- 취향/흥미가 없어도 연결되고 싶다

---

## Our Solution
1. Big Five 성격 + 관심사 기반으로 잘 맞는 사람을 실시간 매칭
2. 두 사람의 현재 위치를 복합 계산해 최적 중간 약속 장소 자동 제안
3. GPS/BLE/UWB로 실제 만남 도착 확인
4. 만남 후 롤링페이퍼 — 상대에 대한 기억을 디지털 피규어로 수집

"앱이 장소까지 잡아준다" — 이것이 허들을 제거하는 핵심 가치 제안.

---

## MVP Features (exactly 3)
1. **Big Five + 관심사 기반 매칭** — 성격 궁합 알고리즘으로 어색함 최소화
2. **약속 장소 자동 제안** — 두 사람의 위치를 복합 계산해 최적 중간 카페/장소 추천
3. **만남 후 롤링페이퍼** — 만남 기록을 남기는 감성적 피드백 루프 (재방문 동기 부여)

---

## North Star Metric
**MAU 5.8만명** (6개월 목표)
- 부지표: 주간 실제 만남 성사 건수, 롤링페이퍼 작성률

---

## Biggest Risk
**새로운 만남에 대한 심리적 허들**
- 아무리 앱이 편해도 처음 만나는 사람을 만나러 가는 행동 자체가 어렵다
- 완화 전략: Big Five 매칭으로 "잘 맞을 것 같다"는 신뢰 제공 + 장소 자동 제안으로 의사결정 부담 제거 + 안전장치(KYC, 신고 시스템)

---

## Platform
- **Primary**: Mobile App (Flutter — iOS/Android)
- **Secondary**: Web (Next.js — 마케팅/랜딩 + 관리자)

2. Big Five 매칭 알고리즘 검증
3. 약속 장소 추천 로직 (osrm-proxy + 장소 DB) 완성
4. 한국 시장 GTM 전략 수립
5. 레거시 서비스 마이그레이션 완료
